HOW TO ADD GALLERY PHOTOS
=========================

Drop your photos into THIS folder, named exactly:

  01.jpg  02.jpg  03.jpg  04.jpg  05.jpg
  06.jpg  07.jpg  08.jpg  09.jpg

They appear automatically on the Gallery page and the homepage strip.
No code editing needed. A missing file just shows a labelled placeholder.

Tips:
- Landscape photos look best. ~1600px wide is plenty.
- Keep each file under ~500 KB (use an image compressor) so the site stays fast.
- You can use .jpg or rename .png/.jpeg to .jpg.
